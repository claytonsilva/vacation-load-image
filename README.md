# vacation-load-image

Load random image from giphy and return in any request from my server

## Usage

Create `.env` file

```.env
API_KEY="<Your Giphy api key>"
PORT=3000
TAG="<Your tag>"
```

run this project running `yarn` commands

```bash
$ yarn build
âœ¨  Done in 0.71s.

$ yarn start
yarn run v1.15.2
$ node dist/index.js
Vacation app!
```

open on http://localhost:3000/ or http://localhost:<your \$PORT variable>/
